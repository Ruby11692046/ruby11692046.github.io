# DOMINION WAR (1v1 P2P Tetris)

WebRTC (Trystero) を利用した、サーバー不要・ブラウザ完結の1v1対戦型テトリスゲームです。1つの盤面を上下の陣地で共有し、ラインを消すことで相手の陣地を奪い合う陣取りバトルが楽しめます。

## 開発の背景とクレジット

本プロジェクトは、**Antigravity** を用いたバイブコーディングによって実装・生成されました。

また、本ゲームの「白と黒を基調として、画面の上と下で表示しながら行うテトリス」という中核的なゲームデザインは、ヒライスン様が制作された素晴らしいゲーム『**ウラオモテトリス**』を元に、特に許可なくアイデアをそのまま拝借して作成したものです。

# DOMINION WAR (1v1 P2P Tetris)

WebRTC (Trystero) を利用した、サーバー不要・ブラウザ完結の1v1対戦型テトリスゲームです。1つの盤面を上下の陣地で共有し、ラインを消すことで相手の陣地を奪い合う陣取りバトルが楽しめます。

## 開発の背景とクレジット

本プロジェクトは、**Antigravity** を用いた「バイブコーディング (Vibe Coding)」によって実装・生成されました。

また、本ゲームの「1つの縦長盤面を2人で共有し、ラインを消すと相手側の陣地を押し返して領土を奪う」という中核的なゲームデザインは、ヒライスン様が制作された素晴らしいゲーム『**ウラオモテトリス**』を元に、特に許可なくアイデアをそのまま拝借（パクる形でオマージュ）して作成したものです。
